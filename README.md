# Datascope
Absolute System — это десктопное приложение на Python, предназначенное для глубокой проверки юр. лиц и ип. Система автоматизирует сбор данных из государственных реестров и официальных источников, анализирует полученную информацию с помощью нейросетей (LLM) и формирует готовый экспертный отчет в формате
